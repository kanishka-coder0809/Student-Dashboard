🎯 BASIC WORKFLOW (3 Steps)

  1️⃣  SELECT STUDENT
      Click: Dropdown → Choose student by name (Roll No shown)
      
  2️⃣  FILL MARKS DETAILS
      Subject: Choose from 11 subjects
      Marks: Enter 0-100 (Grade auto-calculates!)
      Homework: Complete or Incomplete
      Comments: Optional feedback
      
  3️⃣  CLICK "ADD MARKS"
      ✅ Saved to database
      📊 Appears in table instantly
      📈 Statistics update

⚠️ TIPS & TRICKS

  • Select student first - others depend on it
  • Grade updates immediately when you change marks
  • Comments are optional but helpful for feedback
  • Use precise marks (e.g., 92.5 instead of 92)
  • Always confirm before deleting
  • Multiple subjects can be added for same student
  • Edit button shows pencil icon [✎]
  • Delete button shows trash icon [🗑]

🆘 TROUBLESHOOTING QUICK FIXES

  Problem: Student dropdown empty
  → Add students first from /students page
  
  Problem: Marks not saving
  → Check all required fields filled
  → Check internet connection
  → Refresh page and try again
  
  Problem: Grade not showing
  → Enter marks value 0-100
  → Grade auto-updates, may need refresh
  
  Problem: Can't edit
  → Make sure entry exists in table
  → Try refreshing page
  
  Problem: Delete not working
  → Confirm the confirmation dialog
  → Entry removed immediately after
