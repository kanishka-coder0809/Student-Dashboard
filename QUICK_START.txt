STEP-BY-STEP WORKFLOW
═══════════════════════════════════════════════════════════════════════════════

STEP 1: Create Classes
────────────────────────────────────────────────────────────────────────────────
  1. Navigate to http://localhost:3000/classes
  2. Click "Add New Class" button
  3. Enter class details:
     - Class Name: "10-A" (required)
     - Section: "Science" (optional)
     - Description: "Class details..." (optional)
  4. Click "Add Class"
  
  → Classes now appear in dropdown when adding students

EXAMPLE CLASSES TO CREATE:
  • 10-A, 10-B, 10-C
  • 11-A, 11-B (Science, Commerce sections)
  • 12-A, 12-B

─────────────────────────────────────────────────────────────────────────────

STEP 2: Add Students
────────────────────────────────────────────────────────────────────────────────
  1. Go to http://localhost:3000/dashboard
  2. Click "Add Student" button
  3. Fill in student form:
     - Full Name: "Aarjun Kumar" (required)
     - Roll No: "101" (required)
     - Class: Select from dropdown (required)
     - Month: Select month for attendance (required)
     - Monthly Attendance: 85% (required)
  4. Click "Add Student"
  
  NOTE: Email field is NO LONGER needed
        Use Monthly Attendance instead of global percentage

EXAMPLE STUDENT ENTRY:
  Name: Aarjun Kumar
  Roll No: 101
  Class: 10-A
  Month: April
  Attendance: 92%

─────────────────────────────────────────────────────────────────────────────

STEP 3: Add Marks
────────────────────────────────────────────────────────────────────────────────
  1. Go to http://localhost:3000/marks
  2. Select a student from dropdown
  3. Fill in marks form:
     - Subject: Select from list (English, Math, Science, etc.)
     - Marks: 0-100
     - Grade: Auto-calculates! (92 → A+)
     - Homework: Complete/Incomplete
     - Comments: Optional feedback
  4. Click "Add Marks"
  
  GRADE SYSTEM (Auto-Calculated):
     90-100 → A+ (Excellent)
     85-89  → A  (Very Good)
     75-84  → B  (Good)
     65-74  → C  (Satisfactory)
     55-64  → D  (Acceptable)
     0-54   → F  (Fail)

─────────────────────────────────────────────────────────────────────────────

STEP 4: Export Data
────────────────────────────────────────────────────────────────────────────────

OPTION A: Export All Students (PDF Report)
  1. Go to Dashboard: http://localhost:3000/dashboard
  2. Click "Export PDF" button (top right)
  3. File downloads: "student_management_report.pdf"
  4. Contains: All students, marks, attendance, averages

OPTION B: Export Individual Student (PDF Profile)
  1. Go to Student Detail: http://localhost:3000/students/[student-id]
  2. Click "Export PDF" button (top right)
  3. File downloads: "[Student-Name]_report.pdf"
  4. Contains: Full student profile, all marks, attendance details

FIELD DEFINITIONS
═══════════════════════════════════════════════════════════════════════════════

STUDENT FIELDS
  • Name: Student full name (required)
  • Roll No: Student ID/number (required)
  • Class: Select from created classes (required)
  • Month: Month for attendance record (required)
  • Attendance %: Monthly attendance 0-100 (required)

CLASS FIELDS
  • Class Name: Unique class identifier (required) - e.g., "10-A", "11-Science"
  • Section: Optional section type - e.g., "Science", "Commerce", "Arts"
  • Description: Optional notes about the class

MARKS FIELDS
  • Subject: Select from 11 subjects (required)
  • Marks: 0-100 score (required)
  • Grade: Auto-calculated from marks
  • Homework: Complete/Incomplete status
  • Comments: Optional teacher feedback

TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════════════════

Problem: Classes dropdown empty when adding student
Solution: First create classes in /classes page, then add students

Problem: Attendance not saving
Solution: Make sure month is selected and attendance is 0-100

Problem: Grade not calculating
Solution: Marks must be 0-100, grade calculates automatically

Problem: Export PDF not working
Solution: Check browser allows downloads, ensure student has marks

Problem: Can't see created classes
Solution: Refresh page (Ctrl+R), classes fetch from database
