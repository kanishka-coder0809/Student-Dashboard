DATABASE SCHEMA
═══════════════════════════════════════════════════════════════════════════════

STUDENTS TABLE (UPDATED - NO EMAIL):
  id (BIGSERIAL) - Primary key
  name (VARCHAR) - Student name
  roll_no (VARCHAR) - Roll number (UNIQUE)
  class (VARCHAR) - Class name from classes table
  attendance_percentage (DECIMAL) - Monthly attendance %
  created_at (TIMESTAMP) - Auto-created
  updated_at (TIMESTAMP) - Auto-updated

MARKS TABLE:
  id (BIGSERIAL) - Primary key
  student_id (BIGINT) - Foreign key to students (CASCADE DELETE)
  subject (VARCHAR) - Subject name (11 predefined subjects)
  marks_obtained (DECIMAL) - Marks (0-100)
  grade (CHAR) - Auto-calculated grade (A+, A, B, C, D, F)
  homework_status (VARCHAR) - Complete/Incomplete
  teacher_comments (TEXT) - Optional comments
  created_at (TIMESTAMP) - Auto-created
  updated_at (TIMESTAMP) - Auto-updated

CLASSES TABLE (NEW):
  id (BIGSERIAL) - Primary key
  class_name (VARCHAR) - Unique class name (e.g., "10-A")
  section (VARCHAR) - Optional section (e.g., "Science", "Commerce")
  description (TEXT) - Optional class description
  created_at (TIMESTAMP) - Auto-created
  updated_at (TIMESTAMP) - Auto-updated

MONTHLY_ATTENDANCE TABLE (NEW):
  id (BIGSERIAL) - Primary key
  student_id (BIGINT) - Foreign key to students (CASCADE DELETE)
  month (VARCHAR) - Month name (January-December)
  year (INTEGER) - Year
  attendance_percentage (DECIMAL) - Monthly attendance %
  created_at (TIMESTAMP) - Auto-created
  updated_at (TIMESTAMP) - Auto-updated
  UNIQUE CONSTRAINT on (student_id, month, year)

Indexes:
  • idx_marks_student_id on marks(student_id)
  • idx_students_class on students(class)
  • idx_students_roll_no on students(roll_no)
  • idx_monthly_attendance_student_id on monthly_attendance(student_id)
  • idx_monthly_attendance_month_year on monthly_attendance(month, year)
  • idx_classes_class_name on classes(class_name)
